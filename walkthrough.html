Live path

Fastest live demo sequence

  1. Private anchor: frame the protected internal-AI story.
  2. Show the lanes: explain trust and capability as product choices.
  3. Open review: show inside versus outside boundary clearly.
  4. Run result: prove provenance, audit, and labeled return flow.
  5. Switch to Private+: show reduced payload and approval gating.
  6. End on Deterministic: reinforce the trust anchor and fact-first lane.

Cue card

30-second presenter summary

Consentext AI Gateway is a governed product layer for sensitive health AI. The core idea is that the user chooses the trust boundary before any AI step begins. The four lanes change privacy boundary, approval, payload handling, and result behavior. Consentext stays in the middle by keeping review, controls, provenance, and audit visible instead of handing the user directly to an outside AI interface.
